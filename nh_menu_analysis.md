# NH농협 올원뱅크 — 메뉴 구조 분석 & 배리어프리 에이전트 스코핑
> 기준: 금융상품 > 퇴직연금 > IRP 신규가입/입금 메뉴 트리
> 관점: 노년층(디지털+금융 취약) / 주린이(금융 개념 취약)

---

## 1. 메뉴 트리 전체 구조

```
금융상품
├── 대출
├── 퇴직연금 ★ 에이전트 핵심 타겟
│   ├── MY퇴직연금
│   ├── IRP 신규가입/입금 ★★ 최우선 시나리오
│   │   ├── [퇴직연금 가입]
│   │   │   ├── 개인형 IRP 세액공제용
│   │   │   ├── 개인형 IRP 퇴직금 수령용
│   │   │   └── 확정기여형 퇴직연금 (DC)
│   │   ├── [운용상품 조회]
│   │   │   ├── 예금
│   │   │   ├── 펀드
│   │   │   ├── ETF
│   │   │   └── 디폴트 옵션
│   │   └── [퇴직연금 관리]
│   │       ├── MY퇴직연금
│   │       ├── 포트폴리오
│   │       └── 연금설계
│   ├── 운용상품관리
│   ├── ETF 당일 매매/관리
│   ├── 연금계좌관리
│   ├── 상품안내/설계
│   └── 알림 및 고객지원
├── 외환
├── 보험
├── 펀드
├── 신탁
├── 금융상품비교
├── 영업점상품관
└── 스마트상담센터
```

## 2. 진입 경로 분석 (클릭 뎁스)

| 목표 태스크 | 경로 | 클릭 수 | 난이도 |
|---|---|---|---|
| IRP 세액공제용 가입 | 금융상품 → 퇴직연금 → IRP신규가입 → 퇴직연금가입 → 개인형IRP세액공제용 | **5단계** | 🔴 높음 |
| ETF 운용상품 조회 | 금융상품 → 퇴직연금 → IRP신규가입 → 운용상품조회 → ETF | **5단계** | 🔴 높음 |
| 내 연금 현황 보기 | 금융상품 → 퇴직연금 → MY퇴직연금 | **3단계** | 🟡 중간 |
| 포트폴리오 확인 | 금융상품 → 퇴직연금 → IRP신규가입 → 퇴직연금관리 → 포트폴리오 | **5단계** | 🔴 높음 |
| 연금설계 | 금융상품 → 퇴직연금 → IRP신규가입 → 퇴직연금관리 → 연금설계 | **5단계** | 🔴 높음 |

> **핵심 문제**: "IRP 신규가입/입금" 메뉴 안에 [가입], [운용상품 조회], [관리]가 혼재
> → 이름만 보면 "신규 가입하러 들어왔다가" 관리 기능이 있다는 걸 모름 (정보 구조 혼란)

## 3. 배리어프리 취약 구간 — 페르소나별

### 👴 노년층 (디지털 + 금융 취약)

| 구간 | 취약 이유 | 에이전트 개입 포인트 |
|---|---|---|
| `금융상품` 진입 | 홈에서 어디 눌러야 할지 모름 | "퇴직연금 납입하러 오셨나요? 아래 **금융상품** 메뉴를 눌러주세요" |
| `퇴직연금` 하위 메뉴 나열 | 7개 항목이 한 번에 펼쳐져 압도감 | "여기서는 **MY퇴직연금**만 보시면 돼요. 나머지는 지금 안 누르셔도 됩니다" |
| `IRP 신규가입/입금` 선택 | "신규가입"인데 왜 내 계좌가 보이지? 혼란 | "이미 IRP 계좌가 있으시면 여기서 입금도 할 수 있어요. 새로 만드시는 게 아니에요" |
| `개인형 IRP 세액공제용` vs `퇴직금 수령용` 구분 | 둘 다 IRP인데 뭐가 다른지 모름 | 두 종류 차이를 1문장으로 설명 후 추천 |
| `디폴트 옵션` | 단어 자체를 모름 | "'디폴트 옵션'은 제가 직접 고르지 않아도 알아서 굴려주는 상품이에요" |

### 🐣 주린이/사회초년생 (금융 개념 취약)

| 구간 | 취약 이유 | 에이전트 개입 포인트 |
|---|---|---|
| `DC` vs `IRP` 구분 | 회사 다니면 DC, 개인이면 IRP — 모름 | 고용형태 질문 후 적합한 유형 안내 |
| `운용상품 조회` — 예금/펀드/ETF/디폴트옵션 | 4가지 중 뭘 골라야 하는지 판단 불가 | 위험 성향 간단 질문 → 적합 상품군 설명 (권유 아닌 설명) |
| `ETF 당일 매매/관리` | "당일 매매"의 의미, 시장가/지정가 혼란 | ETF 매매 기초 용어 RAG 설명 |
| `포트폴리오` | 내 연금이 어떻게 굴러가고 있는지 해석 불가 | 수익률·위험지표 시각적 해설 |
| `연금설계` | 은퇴까지 얼마나 쌓아야 하는지 계산 개념 없음 | "지금 월 얼마 넣으면 나중에 얼마 받는지" 시뮬레이션 설명 |

## 4. 에이전트 커버리지 우선순위

```
★★★ 최우선  IRP 신규가입 플로우 (세액공제용 vs 퇴직금수령용 분기)
★★★ 최우선  운용상품 선택 (예금/펀드/ETF/디폴트옵션 설명)
★★  중요     MY퇴직연금 현황 해설 (수익률·위험 지표)
★★  중요     디폴트 옵션 개념 설명
★   추후     ETF 당일 매매/관리
★   추후     연금설계 시뮬레이션 해설
```

## 5. 골든 셋 시나리오 초안 (Phase 3 테스트용)

| # | 사용자 발화 | 페르소나 | 기대 에이전트 동작 |
|---|---|---|---|
| S1 | "IRP 납입하러 왔어요" | 노년층 | 화면 네비게이션 → 금융상품 > 퇴직연금 > IRP신규가입 단계 안내 |
| S2 | "세액공제용이랑 퇴직금수령용이 뭐가 달라요?" | 주린이 | 용어_설명_tool(RAG) → 1문장 차이 설명 + 고용형태 확인 질문 |
| S3 | "디폴트 옵션이 뭔가요?" | 노년층 | 용어_설명_tool → 쉬운 말로 설명 ("알아서 굴려주는 옵션") |
| S4 | "ETF 사야 하나요?" | 주린이 | guardrail_tool 감지 → 설명으로 전환 ("ETF가 뭔지 먼저 설명해 드릴게요") |
| S5 | "내 연금이 잘 굴러가고 있는지 보고 싶어요" | 주린이 | 화면_네비게이션_tool → MY퇴직연금 > 포트폴리오 안내 + 수익률 해설 |
| S6 | "여기서 뭘 눌러야 해요?" (퇴직연금 메뉴 진입 후) | 노년층 | 현재 화면 파악 → 목적 질문 → 해당 항목만 지목해서 안내 |
